Import & Export Data

Transfer your health data between devices or keep a backup

Chronic Health stores all data locally on your device — there's no cloud sync or account system. The import/export feature lets you move your data manually using a standard JSON file.

Do I need this?

In most cases, you don't. If you use iCloud Backup (the default on most iPhones), your entire device — including all Chronic Health data — is automatically backed up to your iCloud account. When you set up a new iPhone and restore from that backup, your Chronic Health data comes with it, no extra steps needed.

You can check whether iCloud Backup is enabled in Settings > [your name] > iCloud > iCloud Backup.

The import/export feature is useful when iCloud Backup isn't an option — for example, if you don't use iCloud, if you're setting up a new phone as a fresh device instead of restoring from backup, or if you want an extra manual backup for peace of mind.

What's included in the export?

An export captures everything in the app:

  • Configurations — all your symptoms, treatments (with side effects), and illnesses (with symptom associations)
  • Survey history — every daily survey you've completed, with all recorded values
  • Settings — your daily reminder time and whether reminders are enabled

How to export

  1. Go to the Settings tab
  2. Tap Export Data
  3. Choose where to save the file (iCloud Drive, Files, AirDrop, etc.)

The file is saved as a .json file that you can store, share, or transfer however you like.

How to import

  1. Go to the Settings tab
  2. Tap Import Data
  3. Select a previously exported file
  4. Confirm when prompted

Important: Importing replaces all existing data in the app. Any data currently in the app will be overwritten. Export your current data first if you want to keep it.

Common use cases

Switching to a new phone

If you're moving to a new iPhone and don't use iCloud Backup (or want to be safe), export your data on the old device, transfer the file (via AirDrop, iCloud Drive, email, etc.), and import it on the new device.

Keeping a manual backup

Periodically export your data and save the file to iCloud Drive or another location. If anything happens to your device, you can restore your full health history.

Privacy note: The export file contains your health data in plain text. Treat it like any other sensitive personal information. Chronic Health never transmits this file — you control where it goes.

Troubleshooting

Import doesn't seem to work

Make sure you're selecting a file that was exported from Chronic Health. The file should be in JSON format. If the file is corrupted or from a different app, the import will show an error.

I accidentally imported and lost my data

If you had previously exported your data, you can re-import that earlier export file. If not, the overwritten data cannot be recovered — this is why we recommend keeping periodic backups.
